# Expense Tracker

A comprehensive Flutter application designed to help users track and visualize their personal expenses efficiently.

## Features

- **Track Expenses**: Add detailed records of your daily expenses including title, amount, date, and category.
- **Categorization**: Organize expenses into categories like Food, Travel, Leisure, and Work.
- **Visual Charts**: View a dynamic chart that breaks down your spending by category.
- **Manage Entries**: Easily remove expenses with swipe-to-dismiss functionality and undo accidental deletions.
- **Responsive Design**: Optimized for both light and dark modes with a modern, adaptive UI.
- **Immersive Experience**: Runs in immersive mode for a focused user experience.

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- [Dart SDK](https://dart.dev/get-dart)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/expense_tracker2.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd expense_tracker2
    ```
3.  Install dependencies:
    ```bash
    flutter pub get
    ```
4.  Run the app:
    ```bash
    flutter run
    ```
